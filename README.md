# Project Nerik

**Project Nerik** is a collaborative passion project inspired by Azurik, Rise of Perithia.  
The goal of this project is to rebuild and reimagine core gameplay concepts while developing our skills as game developers using modern tools.

This project serves as a hands-on learning experience focused on understanding the fundamentals of game development, teamwork, and iterative design.

---

## Project Goals

- Learn the fundamentals of game development in Azurik, Rise of Perithia
- Practice collaboration and version control in a team setting
- Explore core gameplay systems such as movement, environment interaction, and level design
- Gain experience with 3D assets and game-ready models
- Understand the workflow of a real game development project from idea to implementation

---

## Technologies & Tools

- **Game Engine:** Unity  
- **Programming:** C#  
- **IDE:** VSCode 
- **3D Modeling:** Blender  
- **Version Control:** Git & GitHub
